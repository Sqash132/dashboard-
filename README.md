# Credit Risk & Default Prediction Dashboard

A single-file HTML dashboard for portfolio credit-risk analytics, default prediction, exposure analysis, and regional default-rate monitoring.

## Preview

![Dashboard overview](assets/screenshots/dashboard-overview.png)

![Dashboard charts](assets/screenshots/dashboard-charts.png)

## Run locally

Open `Suyash_Gaur_Challenge.html` in Google Chrome.

## Project structure

```text
Suyash_Gaur_Challenge.html   # Self-contained dashboard
assets/screenshots/          # Dashboard previews used in this README
```
## Live Demo

1. Open the live dashboard using this link:
   [View the Credit Risk Dashboard](https://<your-vercel-domain>.vercel.app/Suyash_Gaur_Challenge.html)

2. The dashboard opens directly in Google Chrome or any modern browser.

> Note: Use the complete URL ending in `Suyash_Gaur_Challenge.html`. Opening only the Vercel domain shows a 404 error because the project does not use an `index.html` file.
